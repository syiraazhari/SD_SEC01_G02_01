*Installation Manual*
1. You must download and install Microsoft Visual Studio 2019 or latest
2. You must download and install Wampserver
3. You need to import the alumnicafe.sql database into your localhost using Wampserver
4. Double click on the Cafe Alumni POS System.sln and click Start to run the Alumni Cafe POS System program
